BlogMate Chrome 확장 프로그램 전용

1. 수집하는 데이터

BlogMate 확장 프로그램은 기능 제공을 위해 다음 데이터를 브라우저의 로컬 저장소(chrome.storage.local)에 저장합니다:

  • 답방 큐: 사용자가 추가한 블로그 URL 및 방문 상태
  • 댓글 풀: 사용자가 등록한 댓글 템플릿
  • 설정값: 사용자가 지정한 확장 프로그램 옵션
  • 방문 상태: 답방 완료 여부 표시

모든 데이터는 사용자의 브라우저 내부에만 저장되며 외부 서버로 전송되지 않습니다.

2. 수집하지 않는 데이터

다음 정보는 일절 수집하지 않습니다:

  • 네이버 계정 정보 (아이디, 비밀번호)
  • 브라우징 히스토리 (방문 기록)
  • 개인 식별 정보 (이름, 이메일, 전화번호 등)
  • 위치 정보
  • 금융 정보 (결제 수단, 카드 번호 등)

3. 웹사이트 데이터 접근

확장 프로그램은 네이버 블로그(blog.naver.com, m.blog.naver.com) 페이지에서만 동작하며, 다음 DOM 요소에 접근합니다:

  • 블로그 글 제목 및 블로그 ID (답방 큐 구성)
  • 공감(좋아요) 버튼 영역 (공감 감지)
  • 댓글 입력 영역 (댓글 템플릿 추천)
  • 에디터 영역 (글 입력 도우미, 사용자 요청 시에만)

이 데이터는 확장 프로그램의 기능 제공 목적으로만 사용되며 외부로 전송되지 않습니다.

4. 외부 데이터 전송

확장 프로그램은 기본적으로 외부 서버와 통신하지 않습니다. 단, 다음의 선택적 기능을 사용자가 직접 활성화한 경우에 한해 외부 API와 통신합니다:

  • Gemini API (선택 기능): 댓글 추천 기능 사용 시, 사용자가 직접 입력한 본인의 API 키를 통해 Google Gemini API와 통신합니다. 이 경우 전송되는 데이터는 댓글 생성에 필요한 최소한의 블로그 글 내용입니다.

Gemini API 사용 시 Google의 개인정보처리방침이 추가로 적용됩니다.

5. 데이터 저장 및 보관

모든 데이터는 사용자의 브라우저 내부(chrome.storage.local)에만 저장됩니다. 외부 서버, 클라우드, 데이터베이스에 저장되지 않습니다.

데이터를 삭제하려면 다음 방법을 사용하세요:

  • 확장 프로그램 설정에서 "데이터 초기화" 실행
  • Chrome 설정 → 확장 프로그램 → BlogMate → "사이트 데이터 삭제"
  • 확장 프로그램을 제거하면 모든 데이터가 자동으로 삭제됩니다

6. 데이터 보호

  • 모든 데이터는 사용자 기기의 로컬 저장소에만 보관되어 외부 유출 위험이 없습니다
  • 선택적 외부 API 통신 시 HTTPS 암호화가 적용됩니다

7. 사용자 권리

사용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 저장된 데이터 확인 (확장 프로그램 설정 또는 Chrome DevTools)
  • 저장된 데이터 수정 및 삭제
  • 확장 프로그램 제거를 통한 모든 데이터 완전 삭제

8. 아동 보호

본 확장 프로그램은 13세 미만 아동을 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

9. 방침 변경

본 방침이 변경될 경우, 변경 사항은 이 페이지에 게시되며 시행일이 업데이트됩니다. 중요한 변경 시 확장 프로그램 업데이트 노트를 통해 안내합니다.
